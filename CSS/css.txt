CSS selector:
1. Simple selector: 
    i.Name: html tag/ element name
    ii. Id: '#' 
    iii. Class:

ii. Id selector:
HTML : id as attribute : ex. <p id="myParagraph"> 
css: #myParagraph
Ex.
#myParagraph{
    color:green;
}

iii. Class selector: 
HTML: class as attribute: ex. <p class="myParagraphs">
css : .myParagraphs  
Ex. .myParagraphs{
    color: blue;
}
