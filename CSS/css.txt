CSS selector:
1. Simple selector:

    ii. Id: '#'

ii. Id selector:
HTML : id as attribute : ex. <p id="myParagraph">

    color:green;
}

iii. Class selector: 
HTML: class as attribute: ex. <p class="myParagraphs">
css : .myParagraphs  
Ex. .myParagraphs{
    color: blue;
}
